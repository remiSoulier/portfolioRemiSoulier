Time Mage Tournament (TMT)

Présentation

Time Mage Tournament est un jeu qui a été développé en seulement 24h dans le cadre de la Code Game Jam 2023 organisée par l'IUT de Montpellier-Sète.

Ce jeu est un jeu de combat en 2D isométrique dans lequel le joueur incarne un mage ayant la capacité d'arrêter le temps et qui doit affronter des monstres dans différentes arènes.

Ce jeu a été développé en Lua avec le moteur de jeu Pico-8.

Développement de Pokedle - Application Web Inspirée du Concept pokedle.com

Présentation

Dans le cadre de ma formation en BTS SIO option SLAM, j’ai développé une application web nommée Pokedle. Ce projet étudiant consiste en une reprise du concept du site pokedle.com, adapté et redéveloppé dans un objectif pédagogique afin de concevoir une application complète en architecture MVC.

L’objectif est de faire deviner un Pokémon mystère en un nombre limité d’essais. À chaque proposition, l’application compare les caractéristiques du Pokémon saisi avec celles du Pokémon à trouver (type, génération, taille, poids, stade d’évolution) et fournit un retour visuel. Le projet exploite l’API publique PokéAPI afin de récupérer dynamiquement les données, tout en intégrant une base de données relationnelle pour gérer les joueurs et les parties.
