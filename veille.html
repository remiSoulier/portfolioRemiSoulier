Qu’est-ce qu’une veille technologique ?

La veille technologique consiste à surveiller en permanence les évolutions, les nouvelles technologies et les tendances du domaine du développement web. Elle permet de rester à jour, d’améliorer ses compétences et d’anticiper les changements du secteur.

Thème de ma veille

Ma veille se concentre sur les nouvelles technologies du développement web, notamment les frameworks modernes (React, Vue, Svelte), les outils de déploiement (Docker, Vercel), et les bonnes pratiques en matière de performance et d’accessibilité.

Outils utilisés pour ma veille

  • Feedly – pour suivre les blogs de développeurs et les magazines tech
  • Twitter/X – pour suivre les comptes de développeurs influents
  • GitHub Trending – pour repérer les projets populaires
  • Google Alerts – pour recevoir des actualités sur les mots-clés : “JavaScript”, “Tailwind CSS”, “AI in Web Dev”
  • YouTube (Fireship, Grafikart, The Net Ninja…)
